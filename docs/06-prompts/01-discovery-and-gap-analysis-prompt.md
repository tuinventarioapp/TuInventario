# Prompt de descubrimiento y deteccion de huecos

## Uso

Usar antes de comenzar una fase o modulo nuevo.

## Prompt sugerido

```text
Analiza la documentacion de TuInventario y detecta:
- contradicciones;
- decisiones faltantes;
- riesgos tecnicos;
- dependencias bloqueantes.

Lee al menos:
- 00-master/
- 01-business/
- el directorio tecnico correspondiente a la fase actual.

No implementes codigo todavia.

Entrega:
1. resumen del contexto entendido;
2. lista priorizada de huecos;
3. decisiones que puedes asumir sin preguntar;
4. decisiones que requieren confirmacion por riesgo real;
5. propuesta de orden de implementacion.
```
