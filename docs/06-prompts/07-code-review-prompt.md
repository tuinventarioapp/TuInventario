# Prompt para review tecnico

## Uso

Usar cuando otra IA deba revisar cambios ya implementados.

## Prompt sugerido

```text
Haz una revision tecnica de los cambios de TuInventario con foco en:
- bugs;
- regresiones;
- seguridad;
- inconsistencias de negocio;
- testing faltante;
- documentacion desactualizada.

Prioriza findings sobre resumen.

Revisa especialmente:
- integridad de stock;
- transiciones de prestamos;
- aislamiento por organizacion;
- permisos;
- migraciones y contratos API.

Entrega:
- hallazgos con severidad;
- archivos o zonas afectadas;
- riesgos residuales;
- veredicto corto.
```
