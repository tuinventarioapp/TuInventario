# Historias de usuario MVP

## P0

- `US-001`: como propietario quiero crear mi espacio de trabajo para comenzar a operar mi inventario.
- `US-002`: como admin quiero invitar usuarios con roles para distribuir trabajo sin perder control.
- `US-003`: como gestor quiero crear articulos con categoria, unidad y ubicacion para registrar inventario real.
- `US-004`: como gestor quiero cargar stock inicial para empezar sin usar Excel.
- `US-005`: como gestor quiero registrar entradas, salidas, ajustes y traslados para mantener el inventario al dia.
- `US-006`: como gestor quiero ver el historial de un articulo para entender que ocurrio con el.
- `US-007`: como usuario autorizado quiero solicitar y aprobar prestamos para controlar entregas y devoluciones.
- `US-008`: como admin quiero ver prestamos vencidos para recuperar articulos a tiempo.

## P1

- `US-009`: como usuario quiero recibir alertas de errores, cambios y vencimientos.
- `US-010`: como admin quiero exportar reportes para compartir estado operativo.
- `US-011`: como colaborador quiero buscar rapido por nombre, SKU o categoria para trabajar con velocidad.
- `US-012`: como prestatario quiero revisar mis solicitudes y prestamos sin depender de mensajes informales.

## P2

- `US-013`: como gestor quiero adjuntar imagenes para identificar articulos con menos ambiguedad.
- `US-014`: como admin quiero revisar auditoria de acciones criticas.
- `US-015`: como equipo quiero ver actualizaciones en tiempo real para evitar recargas manuales.
