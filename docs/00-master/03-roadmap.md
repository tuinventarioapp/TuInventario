# Roadmap por fases

## Fase 0 - Fundacion

- documentacion definitiva;
- monorepo;
- Docker;
- PostgreSQL;
- migraciones;
- autenticacion;
- organizacion base;
- observabilidad minima.

## Fase 1 - Core operativo

- catalogos base;
- articulos;
- stock inicial;
- movimientos;
- historial;
- dashboard inicial.

## Fase 2 - Prestamos

- prestatarios;
- solicitudes;
- aprobaciones;
- entrega;
- devolucion;
- vencimientos;
- notificaciones.

## Fase 3 - Cierre de MVP

- reportes CSV y PDF;
- tiempo real;
- pruebas E2E;
- endurecimiento de seguridad;
- despliegue staging y produccion.

## Fase 4 - Evolucion posterior

- app movil;
- QR o codigo de barras;
- alertas avanzadas;
- modo offline;
- automatizaciones mas profundas.
